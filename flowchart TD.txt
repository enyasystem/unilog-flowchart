flowchart TD
    START([ Start]) --> VISIT[User visits InternTrack eLogbook System]
    VISIT --> LOGIN[ Login Page]
    LOGIN --> AUTH[ Authenticate Credentials]
    %% Authentication outcome: valid -> identify role, invalid -> stay on login
    AUTH -->|Authenticated / Valid| ROLE[ Identify User Role]
    AUTH -->|Not Authenticated / Invalid| LOGIN[ Login Page\nShow error & remain on login]
    
    ROLE --> DECISION{ User Role?}
    
    
    DECISION -->| Student| STUDENT[ Student Portal<br/> Log Activities<br/> View Feedback<br/>Track Progress]
    DECISION -->| Supervisor| SUPERVISOR[ Supervisor Portal<br/> Review Logs<br/> Provide Feedback<br/> Approve Activities]
    DECISION -->| Organisation/Company| ORG[ Organisation Portal<br/> View Reports<br/> Track Interns<br/> Generate Analytics]
    DECISION -->| Admin| ADMIN[
     Admin Dashboard<br/> Manage Users & Roles<br/> System Settings<br/> System Analytics]


    STUDENT --> LOG[ Student logs daily/weekly activities]
    LOG --> REVIEW[Supervisor reviews logs]
    REVIEW --> FEEDBACK[ Supervisor gives feedback/approval]
    FEEDBACK --> CHECK{ Internship Complete?}
    CHECK -->| No| LOG
    CHECK -->| Yes| COMPLETE[ Process Complete]
    
   
    SUPERVISOR --> MONITOR
    ORG --> MONITOR
    MONITOR --> MAINTAIN[ System Maintenance & Oversight]
     ADMIN --> MONITOR[ Admin monitors system]

    COMPLETE --> END([End])
    MAINTAIN --> END
